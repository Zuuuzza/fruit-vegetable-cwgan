# Fruit & Vegetable Image Generation with CWGAN-GP

## Overview
This project explores the use of Generative Adversarial Networks (GANs) to generate images of fruits and vegetables.  
The final model is based on a Conditional Wasserstein GAN with Gradient Penalty (CWGAN-GP), which provides more stable training and higher-quality results compared to standard GAN approaches.

The model is trained to generate images of three classes:
- Banana  
- Tomato  
- Cauliflower  

---

## Main goals
- Improve stability of GAN training  
- Generate visually meaningful images  
- Experiment with different GAN variants  

---

## Dataset
The dataset consists of RGB images resized to:

- Resolution: 100 × 100 pixels  
- Classes and counts:
  - Bananas: 77 images  
  - Tomatoes: 347 images  
  - Cauliflowers: 234 images
